## How to run:
### Dev
```
1. npm install
3. fill db credentials in /server/db/dbcredentials.js file and mongo login string using process.ENV.DB_USER / DB_PASSWORD variables in /server/db/db.js 
2. npm run dev
```
### Test
```
(npm install if necessary)
1. npm run test
```
